## BGM 오디오 파일을 여기에 올려주세요

이 폴더에 `.mp3` / `.m4a` / `.ogg` / `.wav` 확장자의 오디오 파일을 업로드하면
자동으로 퀴즈 플랫폼의 BGM 재생 목록에 추가됩니다.

- 파일명은 자유롭게 지어도 됩니다 (예: `bgm1.mp3`, `spring.mp3`)
- 파일을 여러 개 올리면 랜덤 순서로 재생 → 끝나면 다음 곡으로 이어서 재생
- 오디오 파일이 하나도 없으면 우상단 BGM 토글 버튼은 표시되지 않습니다
- 브라우저 정책상 자동재생은 막혀있어서, 사용자가 화면에서 한 번 클릭해야
  첫 곡이 재생됩니다 (토글 버튼이 깜빡이며 안내)
